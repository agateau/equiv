# Create .apk

- Check current branch is master

- Check working tree is clean

- Bump version numbers:
    AndroidManifest.xml versionCode
    AndroidManifest.xml versionName
    Constants

- Check translations are up to date

- Generate signed .apk

- Smoke test

- Commit

- Tag

- Push

# Publish

- Take screenshots

- Upload on Google Play

- Write changelog
